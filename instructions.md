# SKT E-commerce Clothing Website Instructions (Full Guide)

## Tech Stack
- **Frontend:** Use React.js for the user interface.
- **Styling:** Use Tailwind CSS for styling.
- **UI Components:** Integrate Radix UI components for accessibility and usability.

## Core Features
1. **Product Listing:** Display SKT-branded clothing items like T-shirts, pants, and other fashion products.
2. **Shopping Cart:** Allow users to add SKT items to a cart and view their selections.
3. **Payment Integration:** Use Stripe for handling payments securely.
4. **Product Filtering:** Include filters for SKT products by size, color, and category.
5. **User Reviews:** Enable customers to leave reviews and ratings for SKT products.
6. **Wish List:** Allow users to save SKT items to a wish list.
7. **Order Tracking:** Implement order tracking and notifications for SKT orders.

## Structure Overview
- **Home Page:** Display featured SKT products and categories.
- **Product Page:** Detailed view of each SKT item with images, descriptions, reviews, and an "Add to Cart" button.
- **Cart Page:** Summarize selected items with options to adjust quantities or remove items.
- **Checkout Page:** Integrate Stripe for secure payment processing.

## Folder Structure
- **Project Root**
  - **/frontend**: All frontend-related code.
    - **/src**: React components, pages, and styles.
    - **/public**: Static assets like images and fonts.
  - **/backend**: All backend-related code (e.g., Node.js/Express server if applicable).
