# Coding Assistant Rules for Building the SKT E-commerce Website

## 1. Read All Instruction Files First
- Begin by thoroughly reading both `instructions.md` and `ui_design.md`. The first file outlines the overall project features and structure, and the second file guides the visual design and theme.

## 2. Follow the Folder Structure
- Maintain the specified folder structure: place frontend code in `/frontend` and backend code in `/backend`.

## 3. Implement Features Step-by-Step
- Build incrementally, starting with the layout, then core features, and finally additional enhancements.

## 4. Follow the UI Guidelines
- Ensure that the design adheres to the styles and themes outlined in `ui_design.md`, including color schemes, typography, and responsive design.

## 5. Ensure Reusability and Accessibility
- Write modular components and ensure accessibility standards are met as described in the instructions.

By following these rules, you'll ensure a consistent, high-quality, and user-friendly SKT e-commerce website.
