# UI and Theme Guidelines for SKT E-commerce Website

## 1. Overall Theme
- **Brand Identity:** The website should reflect the SKT brand, using a consistent color scheme that matches the logo and brand colors.
- **Minimal and Modern:** Aim for a clean, minimalistic, and modern design that’s easy on the eyes and highlights the products.

## 2. Typography and Colors
- **Fonts:** Use a readable sans-serif font for a modern and clean look (e.g., Inter, Roboto, or Open Sans).
- **Color Palette:** Choose a neutral base (whites, grays) with brand accent colors for buttons and highlights. Ensure high contrast for readability.

## 3. Component Style Guidelines
- **Product Cards:** Each product card should include a high-quality image, product name, price, and a quick “Add to Cart” button.
- **Navigation Bar:** Keep the top navigation bar simple with links to the home page, product categories, cart, and user account.
- **Forms and Buttons:** Use consistent styles for all buttons and form elements. Buttons should have clear hover and active states.

## 4. Responsive Design
- **Mobile-Friendly:** Ensure that the design is fully responsive and works well on mobile devices. Test layouts on different screen sizes.

## 5. Accessibility Considerations
- **Alt Text:** Include alt text for all images to ensure accessibility.
- **Keyboard Navigation:** Make sure all interactive elements can be accessed via keyboard navigation.

Use this file to guide the overall look and feel of the SKT e-commerce site, ensuring a consistent and visually appealing user experience.
